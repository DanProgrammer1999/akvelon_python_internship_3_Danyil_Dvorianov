agentAgentIdTransactionsGet

List all transactions of agent with {agent_id}

URL

https://localhost/agent/{agent_id}/transactions

HTTP Method

GET

Response Type

Parameters

  • Path: agent_id Integer

    Id of agent used to filter the transactions

  • Query: type String

    Specify either income or outcome to receive results with amount greater or less then zero

  • Query: start_date String

    Filter transactions with date after the specified one (inclusive)

  • Query: end_date String

    Filter transactions with date before the specified one (inclusive)

  • Query: sort_by String

    Order results by the specified field in ascending order

  • Query: group_by String

    Apply grouping by a time unit, either `day` or `month`. Return date for each group and total amount for each time period. sort_by parameter does not apply here.
